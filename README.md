# foundry-learn-demo

Monorepo：智能合约（Foundry）+ 前端（Next.js）。

## 目录结构

```
foundry/     # Foundry 合约、脚本、测试、部署记录
frontend/    # Next.js 前端
```

## Foundry

合约开发与部署见 [foundry/README.md](./foundry/README.md)。

```bash
cd foundry
forge build
forge test
```

## Frontend

前端开发与本地联调见 [frontend/README.md](./frontend/README.md)。

```bash
cd frontend
npm install
npm run dev
```
